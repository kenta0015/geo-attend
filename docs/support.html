Common Questions

  • How do I check in to an event?
    Open the event and tap "Check in". Make sure you're within the event's location radius and that location permissions are enabled.
  • What if GPS doesn't work?
    You can use the QR code check-in as a fallback. Ask the event organizer to show the QR code.
  • How do I create an event?
    Switch to "Organize" mode and tap "Create Event". Set the location, date, time, and radius.
  • Location permissions not working?
    Go to your device Settings → GeoAttend → Location, and enable "While Using the App" or "Always".
